## 📦 Installation

```bash
curl -fsSL https://github.com/hieudoanm/free.router/raw/refs/heads/master/packages/cli/bin/fr -o fr
chmod +x fr
sudo mv fr /usr/local/bin/
```

## 🚀 Usage

```bash
fr models
fr run <model>
fr status
```
